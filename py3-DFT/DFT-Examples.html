读入2000-Jan-15的Z500全场数据¶

<matplotlib.collections.QuadMesh at 0x11e0d9a90>

对这个Z500做2D FFT变换¶

注意：这个变换的先验假设并不完美。主要原因是，数据的东西方向是物理上的cyclic，但数据的南北方向并不是物理上的cyclic。这使得南北边界处的差异可能过大，会保留在高频部分的谱系数里。¶

In [6]:
Cz500 = np.fft.rfft2(z500)

# 2D 振幅
Cz500_abs = xr.DataArray(np.abs(Cz500))
Cz500_abs.plot()

如果对谱系数做截断，则返回格点空间后会显得“模糊”，但大尺度特征已被很好滴保留，“抓大放小”¶

Cz500_trunc[:,Ktrunc:] = 0

把照片按 RGB 三色分别进行Fourier变换、截断、再反变换，再三色合成为新图片¶

In [11]:
# RGB 颜色分解
c_red   = pic[:,:,0]
c_green = pic[:,:,1]
c_blue  = pic[:,:,2]

# DFT
C_c_red   = np.fft.rfft2(c_red)
C_c_green = np.fft.rfft2(c_green)
C_c_blue  = np.fft.rfft2(c_blue)

# Truncation
Ktrunc = 500
C_c_red[Ktrunc:,Ktrunc:]   = 0      # 第一个维度也许不该截断
C_c_green[Ktrunc:,Ktrunc:] = 0
C_c_blue[Ktrunc:,Ktrunc:]  = 0

# iDFT
pic_new = pic.copy()
pic_new[:,:,0] = np.fft.irfft2(C_c_red)
pic_new[:,:,1] = np.fft.irfft2(C_c_green)
pic_new[:,:,2] = np.fft.irfft2(C_c_blue)

plt.imshow(pic_new)
imsave('pic_truncated.jpg', pic_new)
